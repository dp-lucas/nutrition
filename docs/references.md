# Research References

Evidence-based citations supporting dietary recommendations in this project.

## Format

Each entry follows this structure:

```
### [#] Short Title
- **Authors:** ...
- **Source:** Journal / Organization, Year
- **Key finding:** One-line summary relevant to this patient
- **Used in:** Which recommendation this supports
```
