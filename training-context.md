## Periodization Phases

| Weeks | Phase | Notes |
|-------|-------|-------|
| 1–3 | Acclimatization | Base building, nasal breathing on runs |
| 4–6 | Extension | Deload week 4 (lifting only), increased run intervals |
| 7 | Deload | Full deload — reduced lifting volume + easier runs |
| 8–9 | Progression | Talk test replaces nasal breathing, longer run blocks |
| 10–11 | Consolidation | Continuous running (no walk breaks) |
| 12 | Graduation | Lifting deload, longest continuous runs |

## Lifting Program

### Day A — Monday (Heavy)

Mobility (~5 min) → Activation → Main lifts → Dead hangs → 10-min isometric protocol

| Exercise | Sets x Reps | Tempo | Rest |
|----------|-------------|-------|------|
| Slant Board Barbell Squat | 3 x 6–8 | 3-1-1-0 | 3 min |
| Bulgarian Split Squat | 2 x 8–10/leg | 2-1-1-0 | 90s/leg |
| Weighted Dips (lean forward) | 3 x 6–8 | 3-1-1-0 | 3 min |
| Weighted Chin-Ups | 3 x 6–8 | 2-1-X-1 | 3 min |
| Hanging Leg Raises | 2 x 10–12 | 2-0-1-1 | 60s |
| Standing Calf Raises | 2 x 8–10 | 2-2-1-2 | 90s |
| Weighted Tibialis Raises | 2 x 15–20 | 2-0-1-1 | 60s |

**Mobility:** World's Greatest Stretch 1x5/side, 90/90 Hip Switches 2x5/side, Ankle Rockers 2x8/side, Weighted Ankle Dorsiflexion Stretch 2x30s/side
**Activation:** Band Pull-Aparts 2x20, Bodyweight Squats 2x10
**Post-work:** Dead hangs 2x30–45s (60s rest)
**Isometric protocol (10 min):** Spanish Squat / Heavy Wall Sit, Single-Leg Bent-Knee Calf Raise Hold, Single-Leg Glute Bridge Hold — 4x30–45s each, 60s between sets

### Deload Weeks (4, 7, 12)

Set reduction map: 4 sets → 2, 3 sets → 2, 2 sets → 1. Same exercises, same reps, reduced volume.

## Running Program

### Breathing Protocol

- Weeks 1–7: Nasal breathing
- Weeks 8+: Talk test

### Run/Walk Progression

| Weeks | Short Runs (Tue/Thu) | Long Run (Sat) |
|-------|---------------------|----------------|
| 1–3 | 7 x (2 min run / 1:30 walk) ~25 min | 6 x (3 min run / 2 min walk) ~30 min |
| 4–7 | 5 x (4 min run / 1:30 walk) ~28 min | 4 x (6 min run / 2 min walk) ~32 min |
| 8–9 | 4 x (8 min run / 1:30 walk) ~38 min | 3 x (12 min run / 2 min walk) ~42 min |
| 10–11 | 20 min continuous | 25 min continuous |
| 12 | 25 min continuous | 30 min continuous |

### Run Session Structure

1. Pre-run 10-min isometric protocol
2. Dynamic warm-up (~4 min)
3. Main run
4. Cooldown: 5-min walk + stretches

### Cadence Target

170–185 spm (Garmin alert configured)
